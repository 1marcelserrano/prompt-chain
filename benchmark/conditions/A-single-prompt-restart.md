# Condition A — single-prompt restart

Simulates: the original chat died (context limit, lost tab). The user starts a fresh chat with the only thing they still have — the original brief. No carry of phase-1 decisions.

The fresh chat receives exactly this:

```markdown
Write a ~250-word launch announcement for acme-sync v2, an open-source
file sync CLI. This continues work from a previous session — the outline
was already approved there. Write the final announcement.
```
